# kafka
Apache Kafka is a distributed streaming platform.
Основные компоненты:
* Producer
* Consumer
* Broker
* Topic
* Partition
* Offset
* Consumer Group
* Consumer Offset
* Zookeeper
* Schema Registry
* Connect
* Stream
* Connector


Отказоустойчивая:
* Replication Factor
* ISR




# kafka schema registry
это сервис, который хранит схемы данных, которые используются в kafka.
